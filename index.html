  • git status: checks what branch we are currently on

  • git checkout branch-name: sets the working branch to the specified branch
  • git add -A: adds all files to the staging area
  • git commit -m "message here": commits the changes to the repo
  • git pull origin main: syncs our verson to the main version
  • git push origin feature/branch: pushes our changes to main
